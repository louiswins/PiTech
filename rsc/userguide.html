πTech Treadmill Simulator Software
Users’ Guide

1 Introduction

The goal of this software product is to simulate the use of a treadmill. The program itself has two main sections:

  1. Treadmill control, which allows you to control the virtual treadmill and runner. This is described in sections 3 and 4.
  2. Workout History, which allows you to view the history of the runner over the course of all workouts. This is described in section 5.

3 Controlling the Belt

There are two main ways to control the belt: manual user control (the default), and with a program. With manual user control, you have full control over the speed and inclination of the treadmill’s belt, while when using a program, the software will control the speed and inclination.

  1. Manual user control, in which the user changes the speed and incline manually as he is using the simulator. The simulated track adjusts according to the updated changes and stay at those settings unless the settings are changed again manually.

  2. Preset workout, a particular pattern that the track follows. The system automatically adjusts the speed and incline at preset intervals to follow that particular pattern. There are a few different patterns that the system can follow.

The preset workout plans are a full-on, immersive workout. Therefore, to apply or remove a program, the treadmill must be stopped.

In addition to controlling the belt, you can set goals for yourself. When you reach the goal, the program will automatically alert you in the Message Area. For more information on setting goals, see section sec:goals.


3.1 Manual User Control

  1. If you are already running under a preset workout plan, you will have to stop the treadmill. Press the Pause/Stop button until the treadmill is fully cleared.

  2. Press QuickStart. The simulator belt will begin moving slowly at the preset speed of 1.

  3. Change the speed by editing the Speed Control. You can use the up and down arrows, or enter the value directly into the text box. The speed has a minimum value of 0 mph, and a maximum of 15 mph. You can adjust it in increments of 0.1 mph.

  4. Change the incline by editing the Incline Control. You can use the up and down arrows, or enter the value directly into the text box. The inclination has a minimum value of 0%, and a maximum of 15%.

  5. When the treadmill is running, pressing Pause will result in the belt slowing down and coming to a stop. The QuickStart/Resume button will display Resume, and the Pause/Stop button will display Stop. Also, a 5:00 timer will appear and start counting down. If Resume is pressed before the 5:00 timer is up, the belt resumes with the same speed and incline it had before Pause was pressed. If the timer runs out, then the treadmill resets.

  6. To resume the treadmill with the same speed and incline when paused, press Resume.

  7. If the user presses Stop while the treadmill is paused, or if the 5:00 timer runs out, then the treadmill will become stopped. When this happens, the speed and inclination are both reset to 0, and any goals or programs are lost. Also, the QuickStart/Resume button displays QuickStart.


3.2 Using a Preset Workout Plan (Automatic Control)

You can use a preset workout plan to let the simulator adjust the speed and incline of the workout. You can still adjust the speed and incline manually during the preset.

There are a few types of workout programs to choose from:

  • Cardio,
  • Cross Country,
  • Sprint, and
  • Weight Loss.

  1. If you are already running under another workout plan, or if you are running manually, you will have to stop the treadmill. Press the Pause/Stop button until the treadmill is fully cleared.

  2. Use the Program Control to select an available program.

  3. All controls in the manual user control, as described in section sec:manual, are also available, but the treadmill will also automatically adjust the speed and incline.


4 Goals

Independent of the type of workout you choose, you have the option of setting a target goal. This is a useful option which will alert you when you have run a certain distance, burned a certain number of calories, or run for a certain length of time.

4.1 Setting Goals

  1. Use the Goal Selection options to select which goal you will work toward: a distance-, duration-, or calorie-based goal.
  2. Use the Goal Target inputs to enter your goal: for distance, how far you want to run; for duration, how much time you want to spend running; or for calorie, how many calories you want to burn.
  3. The value you pick will appear respectively in the Distance, Time, or Calorie Display under Target. If there is no goal of that type active, the Target will display zero.
  4. Press Goal Run Start to activate your goal.
  5. The Goal Run Start button will become an Unset Goal button.

4.2 Changing Goals

At any time during a run, you can adjust your goals using the Goal Selection and Goal Target inputs.

4.3 Unsetting Goals

  1. Select Unset Goal to continue running without the goal.
  2. The Goal Target will clear, and the Unset Goal button will revert to the Goal Run Start button, ready for another goal.
  3. The treadmill will continue moving at the same speed with the same inclination.

4.4 Reaching a Goal

When you reach a goal, the program displays a message in the Message Area. It continues moving at the same speed and incline, and you are free to input another goal.


5 Viewing Workout History

The software stores a detailed history of your workouts and can display several statistics to help you track your fitness over a period of time. In addition to the information displayed in the Treadmill Tab (such as current speed, average speed, current incline, or calories burnt), you can access a long-term history of your runs. This information is available at any time from the History Tab.

5.1 Viewing the Workout History

  1. Select the History Tab.
  2. The treadmill, if moving, will continue at the same speed and incline.
  3. The program will show your user history, including time, distance, and calories burnt per workout.
  4. Select the Treadmill Tab to return to the treadmill control.
